# StudyLens ML

## ML Problem Definition

**Objective:** Predict whether a student will have a highly productive next 7 days.

**Prediction task:** Binary classification (1 = productive, 0 = not productive)

**Inputs:**
- Past study sessions with timestamps
- Duration, focus, notes, subject

**Output:**
- Probability of a productive week

**Assumptions:**
- Past behavior predicts near-term future
- Self-reported focus/energy are reliable

**Limitations:**
- Synthetic data
- External factors not included
